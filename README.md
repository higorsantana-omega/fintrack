## Running Locally

#### Clone repository
```bash
git clone https://github.com/higorsantana-omega/fintrack.git
```

#### Install dependencies
```bash
cd web && npm ci

cd api && npm ci
```

#### Execute web application in development mode
```bash
cd web && npm run dev

cd api && npm run start:dev
```
